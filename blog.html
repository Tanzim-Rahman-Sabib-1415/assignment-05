Common Q&A about DOM !!

What is the Document Object Model (DOM)?

The Document Object Model (DOM) is like a map or structure of an HTML page that the browser uses to understand the content on the page. It represents the entire web page as a tree with branches, where each part of the HTML (like headings, paragraphs, buttons, etc.) is considered an element (or node) on this tree.

When you load a webpage, the browser creates a DOM so you can use JavaScript to interact with the page by changing, adding, or removing content.

Example: The DOM lets JavaScript find a button and make it clickable, change its color, or even remove it.

How do you select an element from the DOM?

You can select an element from the DOM using JavaScript by referencing its tag, class, ID, or attribute. Common methods include:

  • document.getElementById("elementID") - Selects an element by its ID.
  • document.querySelector(".className") - Selects the first element that matches a class, ID, or tag.
  • document.getElementsByClassName("className") - Selects all elements with a specific class name.

Example: To select a button with the ID "myButton":

const button = document.getElementById("myButton");
What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a way to manage events (like clicks) on elements by adding one event listener to a common parent element instead of each child element individually. It works by taking advantage of event bubbling, where events on child elements "bubble up" to parent elements.

Why is it useful?

  • Efficiency: You don’t need to add separate listeners to each element.
  • Handles dynamic elements: It works even if new elements are added to the page later.

Example: A list with event delegation on the parent:


        document.getElementById("parentList").addEventListener("click", function(event) {
            if(event.target.tagName === "LI") {
                alert("You clicked on item: " + event.target.textContent);
            }
        });
How do you manipulate an element's attributes and styles using the DOM?

You can modify an element’s attributes and styles using JavaScript:

To change attributes: Use setAttribute or modify properties directly.


        const image = document.getElementById("myImage");
        image.setAttribute("src", "newImage.jpg");
        // or
        image.src = "newImage.jpg";

To change styles: Use the style property:


        const button = document.getElementById("myButton");
        button.style.backgroundColor = "blue";
        button.style.fontSize = "20px";
